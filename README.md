# OLCAtool-support
Documentation, support and example files for the online OLCA tool

Example files can be found here, in the main repository. To use them you should download and import into OLCAtool - you will need the subscription version to do this. Olcatool-demo does not allow import or export, although it does start with an example already loaded when you access it.

The OLCAtool documentation is in the [wiki](https://github.com/anotherstuart/OLCAtool-support/wiki) and you can propose changes and ask quesitons in [issues](https://github.com/anotherstuart/OLCAtool-support/issues).

Responsive OLCAtool support will be delivered in the [issues](https://github.com/anotherstuart/OLCAtool-support/issues) forum.
